Sniffer Probes

In sniffer mode, you use two serial cables and expose their RX wires (because you want to "receive" data). If the cable is a null modem cable, the TX and RX wires will be crossed, so be aware of that.

Here you see the two probes, each one with an RX and ground wire exposed.

These are then going to a USB-Serial adapter so our modern computers can see these two probes and listen to them from this app.

Using a breadboard, an existing serial connection is exposed and the probes are inserted. An RX wire from each probe taps both data lines via the breadboard. Make sure to wire the ground wires to common ground.

In this image you can see how one could use the probes using a breadboard.

Man in the middle

In this mode, we don't futz with wires and breadboards. Instead, we use two null modem cables and do everything in software.

Now instead of just listening, we're actively responsible for ensuring data gets where it needs to go. If we like, we can alter the data too!
